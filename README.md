## 项目运行
```
npm install
npm run serve
npm run build
```

## 库
* [Vuex](https://vuex.vuejs.org/): Vue.js的官方状态管理方案。
* [Vue-router](https://router.vuejs.org/): Vue.js的官方路由管理器。
