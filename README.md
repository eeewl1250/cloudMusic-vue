<p align="center">基于 Vue2 + Vue-Cli3 + Vuex + Vue-router, <br />仿网易云音乐的播放列表页面和歌曲播放页面(参考安卓版界面)</p>


## 项目运行
```
npm install
npm run serve
npm run build
```

## 库
* [Vuex](https://vuex.vuejs.org/): Vue.js的官方状态管理方案。
* [Vue-router](https://router.vuejs.org/): Vue.js的官方路由管理器。
